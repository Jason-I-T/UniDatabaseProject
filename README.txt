File can be ran by typing './UniversityDB' in the command line.

This program allows a user to save, manage, and create their own university database through a gui.
The gui can take a csv file as an input if the csv file is in the format of test_inputs.csv.

The program is started by clicking the upload, then select test_inputs.csv in the database_files
directory:
    database_files/
    |__
       } test_inputs.csv

Adding - Clicking on the add button will take the user through a serious of prompts to add a new
         person to the database.

Removing - Enter the full name of a person that you wish to delete

Searching - Search for a person based on name (only one search at a time)

The user also has functionality to save and upload their custom university database files.
